# حافظ (Hafiz) - Arabic Vocabulary Flashcards

**Turn Arabic vocabulary screenshots into interactive flashcards.**

Upload a screenshot of your Arabic textbook vocabulary, and Hafiz will automatically extract the words, create flashcard decks, and quiz you in both directions.

---

## ✨ Features

- **📸 Screenshot to Flashcards** — Upload an image of Arabic vocabulary and AI extracts all word pairs
- **📚 Organized by Lesson** — Group flashcards into chapters/lessons for structured learning
- **🔄 Two-Way Quizzing** — Practice English → Arabic or Arabic → English
- **🎲 Random Order** — Cards shuffled to test true memorization
- **📊 Progress Tracking** — See which words you've mastered

---

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/AnasKhan0607/arabic-flashcards.git
cd arabic-flashcards

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env.local
# Add your OpenAI API key to .env.local

# Run locally
pnpm dev
```

Visit `http://localhost:3000` to start learning!

## 📖 How It Works

1. **Upload** — Take a screenshot of your Arabic vocabulary (textbook, PDF, etc.)
2. **Extract** — AI reads the image and extracts Arabic-English word pairs
3. **Organize** — Group words into lessons or chapters
4. **Study** — Flip through flashcards at your own pace
5. **Quiz** — Test yourself with randomized quizzes in both directions
